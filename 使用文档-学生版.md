# My First AI+X Project · 新手上路指南

> 你不需要懂编程,不需要懂 AI。跟着做,15 分钟后你会做出人生第一个 AI 项目。

这份文档带你走两步:
1. **一次性配置**——把 AI 助手(Hermes)装好、把这个引导技能装上(配好就再也不用管)
2. **开始你的第一个项目**——只需要跟它说一句"我是学 XX 的"

---

## 第一部分:一次性配置(约 10 分钟)

### 第 1 步:安装 Hermes

Hermes 是一个开源 AI Agent(由 Nous Research 出品)。
- 官网 / 下载:https://hermes-agent.nousresearch.com
- 使用文档:https://hermes-agent.nousresearch.com/docs
- 源码(GitHub):https://github.com/NousResearch/hermes-agent

打开电脑的「终端」(Mac:聚焦搜索输入 Terminal;Windows:装 WSL 或用 PowerShell),粘贴这行回车:

```bash
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
```

装完后,输入 `hermes` 回车,能进入聊天界面就成功了。

> 卡住了?运行 `hermes doctor` 会自动检查哪里出了问题。更多安装方式见官网文档。

### 第 2 步:连一个大模型(选一个国内能用的)

第一次运行会让你选模型和填 Key。推荐国内可直接访问的:

| 提供商 | 拿 Key 的地方 |
|--------|--------------|
| DeepSeek(便宜好用) | platform.deepseek.com |
| Kimi / 月之暗面 | platform.moonshot.cn |
| 通义千问 / 阿里 | dashscope 控制台 |

运行 `hermes model` 选提供商、填对应的 API Key 即可。

### 第 3 步:装上"第一个 AI 项目"引导技能

这个技能已经放在你的电脑里(`~/.hermes/skills/education/my-first-ai-x-project/`)。
确认它在:

```bash
hermes skills list
```

看到 `my-first-ai-x-project` 就说明装好了。

> 如果没看到,在 Hermes 聊天里输入 `/reload-skills` 刷新一下。

---

## 第二部分:开始你的第一个项目(约 15 分钟)

配置好后,以后每次只要:

### 第 1 步:打开 Hermes

终端里输入:

```bash
hermes
```

### 第 2 步:说出你是谁

在聊天框里直接打一句话,比如:

> 我是学金融的大学生,想做个 AI 项目,但我完全是零基础。

(把"金融"换成你的专业就行。)

### 第 3 步:剩下的交给它

接下来它会:
1. 简单问你几个问题(专业、大几、想花多久、**想怎么学**)——一次只问一个,不知道就说"随便"
2. 真的去网上帮你找几个适合你的项目,**从最简单的排到进阶**
3. 你**回一个数字**选一个
4. 按你选的模式带你做:
   - 🎬 **演示模式**:它直接在这个窗口把项目做给你看,边做边讲 Agent 是怎么"自己动手干活"的
   - 🛠️ **实操模式**:它当教练,你自己开一个窗口亲手指挥 Agent,真正上手
5. 十几分钟后,你就有了第一个能拿出去展示的成果 🎉
6. **接着它还会带你把这个成果做大**:用 Agent 给它加个界面、变成能反复用的小工具,加新功能,甚至封装成你自己的"技能"
7. 最后带你把项目**规范地上传到 GitHub**,变成一个署你名字、能写进简历的公开作品(这一步它会帮你检查版权、隐私、密钥等合规问题,别担心)

> 你用的"Agent"就是 Hermes 本身——它不是只会聊天,它会**自己联网、读文件、写代码、把活干完**。这就是它和普通 AI 聊天的区别。
>
> 小提示:第 6、7 步需要你**真的动手做出东西**才会继续——光听不做它不会带你发布,因为发布的是你真实的作品。

### (实操模式)怎么开第二个窗口

如果你选了实操模式,需要两个窗口:一个是"教练"(问它),一个是"你干活的地方"(指挥 Agent)。
再打开一个终端,输入下面两行——它会建一个共享文件夹,让教练窗能实时看到你的进度:
```
mkdir -p ~/ai-x-workspace && cd ~/ai-x-workspace
hermes
```
> 用 Hermes 桌面版或终端分屏,两个窗口并排看更顺手。
> 让你的 Agent 把成果"存成文件",教练窗那边就能直接看到、帮你点评。

## 你会用到的工具

第一个项目**只用 Hermes**(它自己会联网、读写文件、干活),不用装别的。

到了进阶阶段,它会带你用**扣子 Coze**(coze.cn,国内可访问)——让你自己动手"造"一个能反复用的 AI 小助手。

## 常见问题

**Q:我真的一点代码都不用写吗?**
是的。你要做的是用大白话"指挥"Agent——它给你带填空的模版,你把方括号换成自己的内容就行,句式都不用照抄。

**Q:我怕自己写不好那句"指令"?**
不用怕。它给的每个模版都标好了要填什么;你填完拿不准,可以直接发给它帮你审(它会告诉你哪里写得清不清楚),改好再用。

**Q:它给我的项目和别人不一样?**
对,因为它是根据你的专业实时帮你找的,这才是"AI+X"——AI 加上你的专业。

**Q:结果不满意怎么办?**
它会教你三句"万能追问",把结果改到满意为止。你也可以直接跟它说"这个不好,换一个"。

**Q:卡住了 / 报错?**
终端运行 `hermes doctor`;技能没出现就在聊天里输 `/reload-skills`。

**Q:启动时刷出一大段 conda 报错(An unexpected error has occurred. Conda…)?**
这是你电脑里 Anaconda 的插件问题,不是 Hermes 的错。启动前先运行这一行再开 hermes 就好:
```
export CONDA_NO_PLUGINS=true
```
