# 项目菜单模板与各专业范例

阶段 2 用。把检索结果套进下面的格式,由简到难呈现。

## 标准菜单格式

```
我帮你找了 [N] 个适合你上手的项目,从最简单的开始 👇

🟢【项目 1|最简单|约 10 分钟】<名字>
  做什么:<一句话>
  用什么:Hermes(你的 Agent,会自己联网/读文件/干活)
  产出物:<肉眼可见的东西>
  为什么值得做:<和你专业的关联>

🟡【项目 2|进阶|约 20 分钟】<名字>
  ...

🔴【项目 N|参考向|现在不用动手】<真实开源项目名 ⭐星数>
  这是专业的人怎么玩的,先让你看看,种个草。

想做哪个?回我数字就行(拿不定主意选 1,最快出成果)。
```

## 范例:金融专业(已验证形态)

```
🟢【项目 1|最简单|约 10 分钟】财报"人话翻译机"
  做什么:让 Agent 联网查一家上市公司(如茅台、宁德时代)的最新财报,翻成大白话
  用什么:Hermes(你的 Agent)
  产出物:一页纸的公司财务体检报告
  为什么值得做:面试聊行业时你能说出真东西,不是背百度百科

🟡【项目 2|进阶|约 15 分钟】行业赛道扫描
  做什么:选一个赛道(新能源/白酒/AI芯片),让 Agent 联网扫描主要玩家和最新动态
  用什么:Hermes(你的 Agent)
  产出物:一张行业竞争格局对比表
  为什么值得做:一份像样的行业研究,拿得出手

🟡【项目 3|进阶|约 20 分钟】造一个"投研助手"机器人
  做什么:在扣子上搭一个 Bot,给它公司名它自动查财报+新闻
  用什么:扣子 Coze
  产出物:一个可反复用、可分享的投研机器人
  为什么值得做:简历上能写"独立搭建 AI 投研助手"

🔴【项目 4|参考向|现在不用动手】TradingAgents ⭐90K
  做什么:看看专业的人怎么用多个 AI 协作做交易分析
  为什么放这:让你知道今天玩的东西,专业版长什么样,种个草
```

## 其他专业的第一个项目(🟢 起手式)示例

- **法学**:合同风险清单——上传一份合同,AI 提取关键条款+标红风险点
- **医学/生物**:论文速读笔记——上传一篇文献,AI 拆解方法/结论成结构化笔记
- **理工**:数据一键出图——上传一批数据,AI 清洗+分析+画图
- **文史**:事件时间线——给一个历史事件,AI 梳理成考据清晰的时间线
- **新传/设计**:活动策划案——给一个活动主题,AI 产出完整可落地策划案
- **教育/语言**:双语对照材料——给一段内容,AI 生成教学用双语对照+讲解

每个都遵循同一个心法:**把这个专业里最花时间的一件真实苦活,交给 AI 做,10 分钟出成果。**
