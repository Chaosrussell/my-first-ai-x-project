# 手把手实操脚本(双模式 · 用 Hermes 当 Agent)

阶段 4 用。学生用的 Agent 就是 **Hermes**。根据阶段 0 选的模式走对应脚本。全程用**带标注的模版**(见 prompt-templates.md),不给死 prompt。

---

## 🎬 演示模式脚本(你在本窗口做给他看)

以金融「财报体检」为例,节奏如下:

**开场 · 讲你要干什么 + 亮模版**
> "我现在要让 Agent 去查一家公司的财报。我心里的指令大概长这样(结构)👇
>  ```
>  【你是谁】你是我的金融助手
>  【帮我做什么】联网查【某公司】最新财报
>  【我想知道】1. 它赚不赚钱 2. 靠什么赚 3. 有什么风险
>  【怎么给我】我零基础,用大白话
>  ```
>  看,就是说清楚:让它扮谁、做什么、什么要求。看我来一遍——"

**执行 · 真的调工具**
（真的 web_search / browser 去查,别假装。查 A 股公司真实财报数字时,别死磕 Bing/DuckDuckGo 网页搜索——直接用东方财富 datacenter JSON 接口最稳,配方见 search-playbook.md「工具用不动时怎么办」。）

**点破 Agent 本质**
> "注意看,我没自己编数据——我**真的调了联网搜索**,它自己去查了财报。这就是 Agent:**会自己动手用工具把活干完**,不是光跟你聊天。"

**出成果 + 衔接实操**
> "这就是你的第一份成果 🎉 想不想自己来一遍?回复'实操',我当教练带你亲手做。"

## 🛠️ 实操模式脚本(你当教练,学生另开窗口)

**第 0 步 · 约定共享目录 + 开好两个窗口(一次讲清)**
> "我们这样安排:
>  · **这个窗口**——我当教练的地方
>  · **另一个窗口**——你自己干活、指挥 Agent 的地方
>  为了我能实时看到你的进度,我们共用一个文件夹 `~/ai-x-workspace`。
>  打开一个新终端,输入这两行:
>  ```
>  mkdir -p ~/ai-x-workspace && cd ~/ai-x-workspace
>  hermes
>  ```
>  这样你的 Agent 在这个文件夹里干活,我在教练窗随时能看到它生成的文件。开好跟我说一声。"

**第 1 步 · 给指令卡(带标注模版,不是死 prompt)+ 要求落成文件**
> "第一步:让你的 Agent 去查资料。把【】换成你自己的,**句式不用照抄,能说清楚这几点就行**:
>  ```
>  【你是谁】你是我的【你的专业】助手
>  【帮我做什么】联网查【替换成:你要查的东西】
>  【我想知道】1.【替换成:第一个问题】 2.【替换成:第二个问题】
>  【怎么给我】我零基础,用大白话
>  ```
>  末尾加一句 '**把结果保存成文件放在当前文件夹**',这样我这边就能看到。
>  填好了想让我先看看就发我;有把握就直接贴到你的窗口发送。"

**第 2 步 · 教练窗主动监听(关键)**
学生说"这步做完了" → 你**主动用工具查看**他的产出,不必等他复制粘贴:
- `search_files(pattern="*", target="files", path="~/ai-x-workspace")` 看最新文件
- `read_file(...)` 读内容,基于他真实产出点评
> "我看到你的 Agent 生成 `xxx.txt` 了 👀 读了下,查得挺全,就是【某处】可以再让它展开点。"

**第 3 步 · (可选)审核他的 prompt**
学生发来填好的 prompt → 按五维度点评,只挑 1-2 点最值得改的,给"可以发了"信号。(清单见 prompt-templates.md)

**第 4 步 · 收尾出成果**
> "最后一步,让它把结果整理成正式成果。用这个结构(贴模版 C),填好发给你的 Agent。"

教练窗**全程不代劳执行**——他亲手操作,才有"我在用 Agent"的实感。

## 三句"万能追问"(两种模式通用,教给每个学生)

1. **要更细**:"这个能不能更详细一点,举个具体例子?"
2. **要格式**:"帮我整理成表格 / 做成清单 / 分点写。"
3. **换一遍**:"换一个 [公司/案例/数据/主题] 再做一次。"

## 卡壳急救

| 学生说 | 你怎么办 |
|--------|---------|
| "Agent 说查不到 / 没联网" | 提醒确认联网工具可用;或改成"我把资料贴给它"的方式 |
| "结果好烂 / 不对" | 别让他放弃,用万能追问第 1、3 句重试 |
| "看不懂它说的" | 教他直接回 Agent:"这句我看不懂,用更简单的话再说一遍" |
| "两个窗口搞混了" | 重申:教练窗=问我,干活窗=指挥 Agent;建议分屏或桌面版 |
| "下一步呢" | 立刻给下一步,保持节奏,别让他等 |
