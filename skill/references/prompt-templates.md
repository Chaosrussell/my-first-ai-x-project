# 提示词模版库(动态匹配 + 标注填空 + 审核)

阶段 4 用。核心原则:**不给学生写死的 prompt,而是根据"当前项目 + 当前进度"给一个带标注的模版,让他自己填。**

## 素材来源(真实、可给学生的权威仓库)

这些是搜到的高质量 prompt 资源,可作为链接推荐给想深入的学生,也是本文件模版的设计依据:

- **LangGPT**(⭐12K,结构化提示词框架的提出者,中文原生)——本文件模版结构的来源
  https://github.com/langgptai/LangGPT
- **awesome-chatgpt-prompts-zh**(⭐60K,MIT,中文场景大全)
  https://github.com/PlexPt/awesome-chatgpt-prompts-zh
- **prompts.chat / Awesome ChatGPT Prompts**(⭐164K,英文社区最大)
  https://github.com/f/prompts.chat

给学生时用**链接**引导("想看更多模版可以逛这个仓库"),不要整包塞给他。

---

## 模版怎么给:三条铁律

1. **按项目和进度匹配**——检索类任务给"检索模版",分析类给"分析模版",生成类给"生成模版"。不要一个模版走到底。
2. **标注需要填的地方,并说明可自由改**——用 `【替换成:…】` 标注,并明确告诉学生:
   > "这只是个参考结构,你**完全可以不按这个句式**,用自己的话说清楚这几件事就行。"
3. **支持审核**——学生填完后如果不确定,可以把 prompt 发回来,你按"审核清单"给他反馈。

## 按进度分层的模版

### 模版 A · 检索/调研类(项目第一步常用)

```
【你是谁】你是我的 [专业] 助手
【帮我做什么】联网查一下 【替换成:你要它查的东西,比如"宁德时代最新财报"】
【我想知道】
  1. 【替换成:你最想搞清楚的第一个问题】
  2. 【替换成:第二个问题】
【怎么给我】我是零基础,请用大白话,别堆术语
```

### 模版 B · 分析/整理类(拿到材料后)

```
【你是谁】你是我的 [专业] 助手
【这是材料】【替换成:粘贴你的资料,或让它用上一步查到的】
【帮我分析】【替换成:你要它分析出什么,比如"这家公司的风险点"】
【给我什么形式】【替换成:表格 / 清单 / 分点,选一个】
```

### 模版 C · 生成/成果类(收尾出东西)

```
【你是谁】你是我的 [专业] 助手
【帮我做出】【替换成:你要的成果,比如"一页纸的财务体检报告"】
【要包含】【替换成:你希望里面有哪几块内容】
【风格】给一个能直接拿去 [替换成:面试/交作业/展示] 用的版本
```

**给模版时的话术示范:**
> "这一步我们要让 Agent 去查资料,给你个参考结构 👇 你把【】里的换成你自己的,**句式不用照抄,能说清楚这几点就行**:
> (贴模版 A)
> 填好了可以直接发给 Agent;要是拿不准填得好不好,先发给我,我帮你看看。"

## 审核清单(学生把填好的 prompt 发回来时,你按这个点评)

逐条检查,用鼓励口吻反馈,指出 1-2 个最值得改的点即可,别一次挑一堆毛病:

| 维度 | 看什么 | 反馈示范 |
|------|--------|---------|
| **清晰** | 有没有说清要 Agent 干什么 | "任务写得很清楚 👍" / "'分析一下'有点笼统,补一句你想分析出什么会更好" |
| **具体** | 有没有具体对象/范围 | "建议把'某公司'换成具体公司名,结果会准很多" |
| **有条理** | 要求是不是分点/有结构 | "把两个要求分成 1、2 两点,Agent 更好抓重点" |
| **可执行** | Agent 能不能做到(别让它做办不到的) | "这个需要实时数据,记得让它联网哦" |
| **说明到位** | 有没有交代"我是谁、要什么形式" | "加一句'我是零基础'能让它说得更好懂" |

**审核完给一个"可以发了"的明确信号:**
> "改完这一点就很棒了,发给 Agent 试试吧!结果出来我们再一起看。"

---

## 教学生"从模版毕业"

- **第一个项目**:给模版 + 标注 + 可选审核(托底)
- **追问阶段**:只给模版骨架,不标注,让他自己填
- **收尾时点破**:
  > "发现没?指挥 Agent 其实就是说清楚四件事:你要它扮谁、做什么、有什么要求、给什么形式。记住这个,你以后干任何事都能指挥 AI 了。"
