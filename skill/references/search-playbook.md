# 检索配方:如何为不同专业找到"零基础可上手"的 AI 项目

这份文档告诉你(AI)在阶段 1 具体怎么搜、搜什么、怎么过滤。

## 通用检索流程

1. **先用 web_search** 搜中文优先的入口(覆盖面最广):
   - `[专业] AI 能做什么 项目`
   - `[专业] AI 工具 推荐 2025`
   - `[专业] 大学生 AI 入门 案例`
   - `[专业] + AI + no code / 不用编程`

2. **再用 GitHub API 补真实项目**(可选,给"参考种子"用):
   ```
   curl -s "https://api.github.com/search/repositories?q=<关键词>&sort=stars&order=desc&per_page=10"
   ```
   解析出 `full_name` / `stargazers_count` / `description` / `html_url`。

3. **过滤**——GitHub 结果里大量是需要编程的重型项目。判断标准:
   - README 里出现 "pip install" / "clone" / "运行环境" / "Python" → 归为 🔴 参考向,不作第一个项目
   - 是"提示词合集""工作流模板""无代码工具"→ 可作 🟢🟡
   - 中文无关噪音(书单、政治仓库)→ 直接丢弃

## 工具用不动时怎么办(实测优先用结构化 JSON 接口,别死磕网页搜索)

实测:用 browser 打开 Bing / DuckDuckGo 的网页搜索经常**拿不到结果**——Bing 是 SPA,快照常常一片空白;DuckDuckGo 会弹**验证码(CAPTCHA)**。别在网页搜索上反复重试浪费轮次,直接改用**返回干净 JSON 的接口**,这些既稳又能拿到真实数字/星数:

- **找 GitHub 项目 → GitHub Search API**(无需登录,直接出真实 star 数):
  `https://api.github.com/search/repositories?q=<关键词>&sort=stars&order=desc&per_page=8`
  用 browser 打开后,在 console 里 `JSON.parse(document.body.innerText).items.map(r=>({name:r.full_name,stars:r.stargazers_count,desc:r.description,url:r.html_url}))` 解析。
  (已验证真实结果:TauricResearch/TradingAgents ⭐90K、hsliuping/TradingAgents-CN ⭐29K,可作金融 🔴 参考种子。)

- **查 A 股上市公司真实财报(演示模式「财报体检」必用)→ 东方财富 datacenter JSON 接口**:
  `https://datacenter-web.eastmoney.com/api/data/v1/get?reportName=RPT_LICO_FN_CPD&columns=ALL&filter=(SECURITY_CODE%3D%22300750%22)&sortColumns=REPORTDATE&sortTypes=-1&pageSize=3&source=WEB&client=WEB`
  把 `300750` 换成目标股票代码。`columns=ALL` 最省事(字段名易记错,直接全拿再挑)。返回字段:`REPORTDATE` 报告期、`TOTAL_OPERATE_INCOME` 营收、`PARENT_NETPROFIT` 归母净利、`BASIC_EPS` 每股收益、`WEIGHTAVG_ROE` 净资产收益率、`YSTZ` 营收同比、`SJLTZ` 净利同比。
  注意:`push2.eastmoney.com` 的行情快照接口常超时,别依赖它;datacenter 接口够出一份财务体检报告了。

## 冷门 / 无现成项目专业:痛点诊断三层法

当某专业**搜不到零基础可上手的项目**(0 结果,或全是要编程的重型研究代码)时,**别硬塞重型项目**。改从学生的真实困境里长出项目方向。层层递进,始终有出路:

### 第 1 层 · 问他自己的困境(先问,别急着给)
> "你学 [专业],平时最头疼、最花时间、最想有人帮一把的是什么事?"

从他说的痛点里找 AI 能发力的点,直接长出项目。多数学生这一层就能给出料。

### 第 2 层 · 他说不出 → Agent 主动启发
学生答"不知道/没什么"时,别停:
- 用你的认知 + **联网查"该专业普遍的学习/研究痛点、日常最耗时的事"**;
- 拿几个常见困境**反问**他,帮他把模糊的难受说成具体痛点:
  > "很多学 [专业] 的同学会卡在这几件事上,你中招哪个?
  >  1️⃣ [该专业常见痛点A]  2️⃣ [常见痛点B]  3️⃣ [常见痛点C]"
- 他一认领,方向就出来了。

### 第 3 层 · 还对不上 → 引导转向相邻角度
把这个专业**拆解到 AI 更成熟的维度**,找"专业 × AI"咬合的接口:
> "纯 [专业] 的现成 AI 场景确实少。那换个角度——你专业里的活儿,更偏哪一块?
>  📄 大量文字/文献 → AI 擅长读和整理
>  🖼️ 图片/影像/实物 → AI 擅长看图、识别、生成
>  📊 数据/统计/测量 → AI 擅长清洗、分析、画图
>  🔁 重复性事务 → AI 擅长自动化、批量
> 我们从你最对得上的那块切进去。"

**⚠️ 边界(务必守住):这是换切入角度,不是劝他离开专业。** 始终锚定在他的领域里,只是找那个和 AI 咬合的接口。例:不是"别学 X 了去搞图像识别",而是"你专业里整理照片/影像这件事,本质是个图像活儿,AI 很擅长,我们从这切入"。

### "痛点 → AI 维度"映射(通用,适配任意专业)
| 学生的痛点长这样 | 对应 AI 维度 | 可长出的项目 |
|-----------------|------------|------------|
| 要读大量长文献/报告/原文 | 📄 文本 | 速读笔记、要点提炼、多篇综述 |
| 看不懂的术语/古文/外文 | 📄 文本 | 术语科普卡、翻译对照、白话解读 |
| 整理零散资料/条目/名录 | 📄 文本 | 结构化资料卡、可查询清单 |
| 一堆照片/影像/样本要归类 | 🖼️ 图像 | 分类整理、特征标注、图文卡片 |
| 一批数据/测量/问卷要处理 | 📊 数据 | 清洗+分析+画图、可视化报告 |
| 反复做同一件事 | 🔁 自动化 | 批量处理、模板生成、定时任务 |

### 必胜兜底(任何情况都能出成果)
如果三层都还没落地,回到最稳的一招——**让学生喂给 Agent 一份他手上的真实材料**(一篇正在读的论文/一份报告/一批数据/一堆条目),让 Agent 十分钟帮他读透、整理、产出成果。**不依赖任何检索,只要学生有一份本专业材料就必然成功。**

**核心心法:第一个项目的价值不是"找到一个专属 AI 项目",而是"让学生看到 Agent 能替他干本专业的真活"。有没有现成 GitHub 项目不重要——每个专业都有值得交给 Agent 的苦活。**

## 相邻专业借力(第3层用)

冷门专业搜不到,但**相近学科往往有共通的活儿和现成项目**,借过来即可。灵活为主——**拿不准就直接问学生"你觉得你专业和哪些方向像?"**,别硬套下面的分组。

| 专业 | 相邻可借的学科 |
|------|--------------|
| 考古 | 历史、人类学、博物馆学、文物保护、地理 |
| 历史/哲学 | 文学、社会学、政治学、文献学 |
| 金融 | 经济、会计、统计、管理 |
| 临床医学 | 护理、药学、公共卫生、生物 |
| 农学 | 生态、环境科学、生物、气象 |
| 法学 | 政治、行政管理、社会学 |
| 建筑 | 城市规划、土木、环境设计、地理 |
| 语言/翻译 | 教育、传播、文学、语言学 |

下面这张表只是起点——**遇到没列的专业,让 Agent 自己发散:"这个专业的日常任务,和哪些学科重叠?"** 找到重叠,就去那些学科的方向里借项目。

## 通用项目库(任何专业都能做,发散 + 按画像推荐)

当专业方向暂时卡住、或学生想先随便玩玩时的优质保底。**这批不是封闭清单——鼓励 Agent 按学生画像(年级/兴趣/就业焦虑等)发散、组合、变体**:

| 项目 | 做什么 | Agent 干活 | 适合 |
|------|--------|-----------|------|
| 📁 文件/资料整理 | 一堆乱文件自动归类、重命名、建目录 | 读文件+批量整理 | 所有人 |
| 📚 个人知识库 | 课件/笔记喂进去,建成可查询的知识库 | 读文件+提炼+组织 | 所有人 |
| 📝 复习作战地图 | 课程/考纲→复习计划+重点+自测题 | 联网+推理+生成 | 期末/备考 |
| 🔍 主题调研报告 | 任何主题→联网汇总成结构化报告 | 联网搜+整理 | 所有人 |
| ✍️ 写作打磨助手 | 简历/文书/演讲稿→润色+结构建议 | 文本改写 | 所有人 |
| 💼 求职弹药库 | 目标岗位→JD分析+面试题+答题思路 | 联网+分析 | 大三大四、就业焦虑 |
| 🗓️ 个人效率工具 | 待办/日程/打卡→自动整理或提醒 | 数据+自动化 | 所有人 |

**发散示例**:面对大一新生可推"知识库+复习";面对大四生可主推"求职弹药库+简历打磨";学生提到某爱好(如摄影/游戏)可即兴组合出相关项目。别拘泥于上表,按人推荐。
